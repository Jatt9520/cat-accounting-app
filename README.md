# 🐱 猫咪记账

一款可爱风格的安卓记账应用，帮助你轻松管理日常收支。

![Platform](https://img.shields.io/badge/Platform-Android%209--15-brightgreen)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ 功能特性

### 📝 收支录入
- 收入/支出一键切换
- 金额快速输入
- 丰富的分类选择（带可爱emoji图标）
- 备注和日期自定义

### 🏷️ 分类管理

| 支出分类 | 收入分类 |
|---------|---------|
| 🍜 餐饮 | 💰 工资 |
| 🚗 交通 | 🎁 奖金 |
| 🎮 娱乐 | 🧸 零花钱 |
| 📚 学习 | 📈 投资 |
| 🛍️ 购物 | 💼 兼职 |
| 🏥 医疗 | 🧧 红包 |
| 🏠 住房 | 💳 退款 |
| 🧴 日用 | 💎 其他 |
| 👥 社交 | |
| 📦 其他 | |

### 📊 数据统计
- **多维度查看**：本周 / 本月 / 本年
- **收支汇总**：总收入、总支出、结余
- **分类占比**：直观的饼图展示
- **趋势分析**：折线图展示收支变化

### 🐱 猫咪弹窗
每次新增账单时，会弹出一只随机小猫，搭配淡入动画，让记账变得更有趣！

- 20种不同猫咪表情
- 10条可爱提示语
- 流畅的弹出动画

### 🎨 界面设计
- 粉白配色，少女心满满
- 圆角卡片，柔和阴影
- 流畅的页面切换动画
- 支持深色模式

## 📱 安装说明

### 下载APK

1. 在本仓库中找到 `cat-accounting.apk` 文件
2. 点击文件，然后点击右上角的 **Download** 按钮下载
3. 将下载的APK传输到安卓手机
4. 在手机上打开文件管理器，找到APK文件
5. 点击安装（如提示需要允许"未知来源"应用安装，请在设置中开启）

> 💡 **提示**：APK文件位于仓库根目录，可直接下载使用

### 系统要求
- Android 9.0 (API 28) 及以上
- 支持 Android 15

## 🚀 本地开发

### 环境准备
- Node.js 16+
- JDK 17
- Android SDK

### 安装依赖
```bash
npm install
```

### 运行开发服务器
```bash
npx cap serve
```

### 构建APK
```bash
npx cap sync android
cd android
./gradlew assembleDebug
```
